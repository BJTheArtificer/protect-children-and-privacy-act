## Why these terms

Legislation should circulate freely. Other states, the federal government, and any jurisdiction working on similar issues should be able to take this work, adapt it, and improve it without licensing barriers.

Commentary and supporting materials are licensed CC BY 4.0 because attribution helps readers track the origin of arguments and analysis as they propagate through subsequent work. If you adapt the plain-English explanation, the fiscal methodology, or the letters, please credit this repository so readers can find the original context.

If you have questions about whether a particular use is permitted, open an issue.
